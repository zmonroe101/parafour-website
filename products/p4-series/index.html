Product Overview

The P4 Series ProCE is an electronic calculator designed for LPG autogas dispensing applications. Running software version 4.45C, the P4 Series is NTEP Category 1 certified — the highest weights-and-measures designation available in the United States — meaning no mechanical seal is required and all calibration and configuration changes are tracked electronically in an event log accessible to any W&M inspector.

Engineered for fleet, autogas, and commercial fueling environments, the P4 Series is compatible with Neptune/Liqua-Tech and Liquid-Controls MA4 series meters and supports both single and dual hose configurations. Automatic Temperature Compensation (ATC) is built in per API Standard 2450 / ASTM D 1250 / ISO R91, correcting volume delivery to 60°F reference temperature for accurate, legally defensible transactions.

The P4 Series is ETL listed and carries CE and GOST certifications for international markets, and meets NEC Class 1, Division I and II hazardous area requirements for installation at LPG fueling facilities.

Key Features

Engineered for the demands of commercial LPG dispensing — from transaction accuracy to W&M compliance.

🌡

Automatic Temperature Compensation

Built-in ATC corrects LPG volume to the 60°F reference standard per API 2450, ASTM D 1250, and ISO R91 — ensuring accurate, temperature-adjusted delivery for every transaction.

💲

Multi-Pricing

Up to 5 configurable price levels selectable via keypad or nozzle switch. Supports fleet pricing, retail pricing, and bulk discount tiers without hardware changes.

🎯

Preset Dispensing

Amount or volume preset entry via keypad or external preset button. Supports single and dual solenoid valve configurations for precise delivery cutoff.

📋

Sales History

Up to 8,000 transaction records stored with timestamp, volume, price, and delivery data. Accessible via keypad or serial port for reconciliation and audit purposes.

📊

Shift Reporting

Attendant login/logoff with shift-level totalizers. Optional receipt printer support via serial port. Shift data supports back-office reconciliation and fuel management system integration.

🔌

Serial Communication

Gilbarco protocol over RS232, RS485, or 2-wire current loop. Configurable baud rate (4800, 5787, or 9600). Enables integration with fuel management systems and back-office software.

💳

Card Reader Integration

MCR authorize signal input with programmable pulse output (1, 10, or 100 pulses per unit volume) and configurable pulse width (0.5–30ms) for compatibility with a wide range of card reader and fleet card systems.

✓

NTEP Category 1 Compliant

The highest W&M designation available. Calibration and configuration event logs are maintained electronically and are accessible to any weights-and-measures inspector — no mechanical seal required.

Menu System

The P4 Series uses a 6-level password-protected menu hierarchy. Each level controls a discrete set of functions, keeping daily operations separate from calibration and W&M-controlled parameters. Full menu documentation is available in the P4 Series Manual.

Attendant

Daily pump operations — transaction entry, price selection, preset input, and start/stop controls.

Manager

Price level configuration, attendant management, shift open/close, and report printing.

Technician

System configuration, serial communication settings, card reader parameters, and solenoid setup.

Calibration

W&M-controlled calibration factors. All changes logged to NTEP event log. Requires W&M inspector access.

Reports

Transaction history, shift totals, calibration event log, and system diagnostic reports.

Master

Full system access, factory default restoration, firmware version display, and diagnostic tools.
